Add Multiple Products

This Application helps to Add Multiple Products in sale, Purchase, Delivery, Invoice only on Draft(New) State.

Basic Features
  • Multi Add Products on Sale Order Line.
  • Multi Add Products on Purchase Order Line.
  • Multi Add Products on Delivery Order Line and Incoming Shipment Line.
  • Multi Add Products on Invoice Line.
